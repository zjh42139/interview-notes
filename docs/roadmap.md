# 复习路线

> 这份文件只回答一个问题：**什么阶段学什么**。另一个问题「什么内容值得学」由 [阅读指南](./阅读指南.md) 回答。两份配合使用——阅读指南决定投入比例，本文件决定学习顺序。

---

## 总周期：10 周 | 每天 2-3 小时

---

### 第一阶段：JS 基础（3 周）

**范围**：[阅读指南 P0 JavaScript](./阅读指南.md#javascript-核心)

**学习路径**：this → 闭包 → 原型链 → Promise → Event Loop → async/await

**完成标准**：

- [ ] 手写 Promise、bind、深拷贝代码能跑通
- [ ] 能在一段话内说清 Event Loop（宏任务→微任务→渲染）的完整顺序
- [ ] `setTimeout(fn, 0)` + `Promise.resolve().then(fn)` 混合执行顺序，任意组合不翻车
- [ ] 同步完成 [面试回答/JavaScript/](面试回答/JavaScript/closure.md) 对应篇的口头练习

---

### 第二阶段：Vue 框架（2.5 周）

**范围**：[阅读指南 P0 Vue3](./阅读指南.md#vue3-核心机制)

**学习路径**：响应式 → Renderer → Scheduler → Diff → 全链路渲染 → computed/watch → nextTick → 生命周期

**完成标准**：

- [ ] 画出响应式原理流程图（track → trigger → effect）
- [ ] 说清 Diff 为什么是 O(n)（只做头对头、尾对尾两趟）
- [ ] 能画出「数据变化 → DOM 更新」的完整链路
- [ ] 手写 nextTick 简化版能跑通
- [ ] 同步完成 [面试回答/Vue3/](面试回答/Vue3/reactivity.md) 对应篇的口头练习

---

### 第三阶段：浏览器 + 网络 + 工程化（2.5 周）

**范围**：[阅读指南 P0 浏览器](./阅读指南.md#浏览器-缓存-渲染-安全) + [P0 网络](./阅读指南.md#网络协议核心) + [P0 工程化](./阅读指南.md#工程化现在每场面试都会问)

**学习路径**：URL 到页面 → 渲染流程 → 关键渲染路径 → 缓存 → 安全 → TCP → HTTP/HTTPS → Vite → Tree Shaking → Code Splitting

**完成标准**：

- [ ] 从 URL 到页面展示，能一口气说清全部阶段（网络 + 渲染 + 合成）
- [ ] XSS/CSRF/CSP 三者的本质区别和防御方式能各用一句话说清
- [ ] 强缓存和协商缓存的 HTTP 头能默写
- [ ] Tree Shaking 的原理能用一句话说清（ESM 静态结构 → 死代码消除）
- [ ] Vite 为什么快——至少说出 3 个原因（ESM 开发/Esbuild 预构建/HMR 热更新）

---

### 第四阶段：冲刺模拟（2 周）

**范围**：[阅读指南 P0 项目实战](./阅读指南.md#项目实战与表达你最值钱的差异化) + [P0 手写题](./阅读指南.md#手写题-p0闭眼能写) + [跨模块知识链](./阅读指南.md#-跨模块知识链p0-核心任务)

**第 1 周：项目实战打通**

- [ ] 项目实战 6 个核心文件 + 面试回答 项目 4 篇——能脱稿讲一个完整的技术决策案例
- [ ] 每个知识点都能落到项目里：你项目里的 Axios 封装怎么做的？权限路由怎么设计的？Token 怎么刷新的？

**第 2 周：手写 + 知识链 + 模拟**

- [ ] P0 手写题全部闭眼能写（任选一题 10 分钟内无 bug）
- [ ] 4 条跨模块知识链——白纸画图，不翻资料
- [ ] 选 1-2 场模拟面试做压力测试
- [ ] HR 模块通读一遍

**完成标准**：

- [ ] 面试自我介绍 1 分钟版和 3 分钟版都能流畅表达
- [ ] 核心 10 道手写题任选一题能 10 分钟内写出无 bug 版本
- [ ] [面试回答](面试回答/) 中至少 10 篇能脱稿复述（录音回听确认）
- [ ] 4 条跨模块知识链——白纸脱稿画出全程，每个箭头旁写出因果关系
