# 面试复习路线

> 总周期：10 周 | 每天 2~3 小时。按阶段推进，每个阶段有"最小集"和"完成标准"。

---

## 按阶段

### 第一阶段：JS 基础（4 周）

**学习路径**：this → 闭包 → 原型链 → Promise → Event Loop → async/await

**如果时间不够，只看这 5 篇**：[this](JavaScript/this.md) · [闭包](JavaScript/closure.md) · [原型链](JavaScript/prototype-chain.md) · [Promise](JavaScript/promise.md) · [Event Loop](JavaScript/event-loop.md)

**完成标准**：
- [ ] 手写 Promise、bind、深拷贝代码能跑通
- [ ] 能在一段话内说清 Event Loop（宏任务→微任务→渲染）的完整顺序
- [ ] `setTimeout(fn, 0)` + `Promise.resolve().then(fn)` 混合执行顺序，任意组合不翻车

---

### 第二阶段：Vue 框架（3 周）

**学习路径**：响应式 → computed/watch → Diff → 生命周期 → KeepAlive → 手写题

**如果时间不够，只看这 4 篇**：[响应式原理](Vue3/reactivity.md) · [Diff/Patch](Vue3/diff-patch.md) · [组件通信](Vue3/component-communication.md) · [nextTick](Vue3/nextTick.md)

**完成标准**：
- [ ] 画出响应式原理流程图（track → trigger → effect）
- [ ] 说清 Diff 为什么是 O(n)（双端比较+最长递增子序列）
- [ ] 手写 nextTick 简化版能跑通

---

### 第三阶段：浏览器 + 网络 + 性能（2 周）

**学习路径**：URL 到页面 → 渲染流程 → 重绘/回流 → 缓存 → 安全（XSS/CSRF/同源）→ 性能优化 → 网络

**如果时间不够，只看这 4 篇**：[URL 到页面](浏览器/url-to-page.md) · [XSS/CSRF](浏览器/安全/xss.md) · [浏览器缓存](浏览器/cache.md) · [首屏优化](性能优化/first-screen.md)

**完成标准**：
- [ ] 从 URL 到页面展示，能一口气说出全部阶段（网络 + 渲染，不含糊）
- [ ] XSS 和 CSRF 的本质区别一句话说清
- [ ] 强缓存和协商缓存的 HTTP 头能默写
- [ ] 能说出至少 3 种首屏优化手段，并解释原理

---

### 第四阶段：冲刺模拟（1 周）

**学习路径**：刷题库 → [面试回答](面试回答/) 逐篇练 → 模拟面试 → 手写题 → HR 准备

**完成标准**：
- [ ] 面试自我介绍 1 分钟版和 3 分钟版都能流畅表达
- [ ] 核心 10 道手写题任选一题能 10 分钟内写出无 bug 版本
- [ ] [面试回答](面试回答/) 中至少 5 篇能脱稿复述（录音回听确认）
- [ ] 模拟面试连续 2 次无卡顿

---

## 按目标

### 社招 P6（核心模块深度 + 项目经验）

```
JS 5 篇 → Vue3 4 篇 → 浏览器 3 篇 → 网络 3 篇 → 性能优化 2 篇
→ 手写题全部 → TypeScript 泛型/工具类型
→ 面试题库你的目标模块 → 项目实战挑 3 个你项目里真正用到的场景
→ 面试回答：JS 6 篇 + Vue3 3 篇 + 项目 4 篇（练脱稿）
→ 算法 30 题（高频题为主）
```

### 校招（基础广度 + 算法 + 手写）

```
HTML 3 篇 → CSS 3 篇 → JS 5 篇 → 浏览器 3 篇 → 网络 2 篇
→ 手写题全部 → 算法模块全部
→ 面试回答：JS 3 篇 + CSS 2 篇 + 浏览器 2 篇
→ HR 准备
```

### 快速突击（两周）

```
首页策展路线「🎯 冲刺」→ 面试题库大模块各刷 5 题
→ 面试回答挑 8 篇核心脱稿练 → 手写题 5 道核心题 → 模拟面试 2 轮
```

---

## 使用建议

- 每个阶段按"最少集"优先——看完一篇就试着口头说一遍，不要追求逐字读完
- 每完成一个阶段，刷对应 [面试题库](面试题库/) 里的 5 道题检验效果
- 遇到答不出来的题，回知识文章补，不要硬背答案
- 面试前 3 天：停止看新文章，专心练 [面试回答](面试回答/) 的脱稿复述
