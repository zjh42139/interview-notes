## 练习建议

1. **P0（必能手写，全部 ⭐⭐⭐⭐⭐）**：Promise / Promise.all / bind / call / new / 深拷贝 / 防抖 / 节流 / curry / 数组扁平化去重 / 并发调度器 / instanceof / EventEmitter / LRU——这 14 道题占手写题考察的 80%。EventEmitter 和 LRU 2026 年追问明显加深（带优先级的 EventEmitter、LRU 的 O(1) 推导），是中大厂的区分题
2. **P1（熟练）**：Promise.race / 列表树互转 / 版本号比较 / LazyMan / 寄生组合继承 / 对象扁平化 / 数组去重（NaN 处理）/ 大数相加 / 虚拟滚动 / CSS 居中 / 圣杯双飞翼——字节/阿里高频
3. **P2（掌握思路）**：compose / pipe / Object.create / JSON.stringify / CSS 三角形 / 移动端 1px / 圆形进度条——考的不多但思路重要

每道题的练习方法：① 脱离文件自己写 → ② 跑测试用例 → ③ 对比实现文件 → ④ 重写直到一次过。**面试时手写题平均 10-15 分钟内完成——平时练习要计时。**

> 手写题不需要回答稿——代码本身就是答案。需要口述原理的见 [JavaScript 面试回答](../面试回答/)（Promise/Event Loop/闭包/原型链等）。
