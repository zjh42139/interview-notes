## 练习建议

1. **P0（必能手写）**：Promise / Promise.all / bind / call / new / 深拷贝 / 防抖 / 节流 / curry / 数组扁平化去重——这 10 道题占手写题考察的 80%
2. **P1（熟练）**：Promise.race / EventEmitter / LRU / 并发调度器 / 列表树互转 / instanceof / 版本号比较 / 对象扁平化——字节/阿里高频
3. **P2（掌握思路）**：compose / pipe——考的不多但思路重要

每道题的练习方法：① 脱离文件自己写 → ② 跑测试用例 → ③ 对比实现文件 → ④ 重写直到一次过。**面试时手写题平均 10-15 分钟内完成——平时练习要计时。**

> 手写题不需要回答稿——代码本身就是答案。需要口述原理的见 [JavaScript 面试回答](../面试回答/)（Promise/Event Loop/闭包/原型链等）。
