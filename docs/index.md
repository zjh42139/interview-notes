## 使用方式

1. **系统复习**：按 [复习路线](./roadmap.md) 四阶段推进
2. **查漏补缺**：搜索知识点，阅读「核心机制」+「深度拓展」
3. **考前冲刺**：刷 [面试题库](./面试题库/) + [模拟面试](./模拟面试/)
4. **手写练习**：独立完成 [手写题](./手写题/) 中的代码实现
