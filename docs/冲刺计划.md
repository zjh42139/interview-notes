# 3 周冲刺计划

> **核心原则：阅读是低 ROI，脱稿复述是高 ROI。** 面试官不听你读了什么，听你能不能 30 秒说清一个原理。
>
> 每天 2 小时：1 小时练说 + 1 小时手写/知识链。不要花时间通读。

---

## 第 1 周：P0 原理 → 脱稿

### 每天 1 个模块

| 天 | 模块 | 练什么 | 怎么练 |
|:---:|------|------|------|
| 1 | **JS 核心** | [P0 JavaScript](./阅读指南.md#javascript-核心) ~13 篇 | 每篇看一遍 → 关掉 → 录音 30 秒讲核心原理 → 回听对比原文。特别练：this 四种规则、原型链查找、闭包应用场景、async/await 输出时序（2 tick 陷阱） |
| 2 | **异步编程** | Promise + async/await + Event Loop + 必问: 手写并发调度器（限制最大并发数） + 手写 Promise | 默写 Promise 完整实现（constructor + then + resolvePromise）——**闭眼能写**才算过 |
| 3 | **Vue3 响应式** | reactivity + computed/watch + nextTick + scheduler + **新增必考**: v-if vs v-show 渲染机制区别（面试第一题） | 白纸画「数据变化 → DOM 更新」完整链路图，不翻资料 |
| 4 | **Vue3 渲染** | Diff/Patch + 全链路渲染 + 编译优化 + Block Tree | 口头讲 Vue3 Diff 五步法 + 为什么不做双端对比 + Block Tree 怎么运作 + 纠正: KeepAlive 是 DOM move 不是销毁重建 |
| 5 | **浏览器** | URL 到页面 + 缓存 + 渲染流程 + XSS/CSRF/CSP | 录音一口气讲完「输入 URL 到页面展示」——控制在 3 分钟内不卡壳 + 强缓存/协商缓存优先级（必考必深挖，有真实丢分案例） |
| 6 | **网络** | HTTP/HTTPS/TLS + TCP + CORS + HTTP2 + **新增必考**: GET vs POST + HTTP 方法幂等性（面试第一题）、CORS 预检三条件 | 画 TCP 三次握手时序图 + TLS 1.2 ECDHE 握手过程 + 说清 CORS 预检触发的三个条件 |
| 7 | **复习** | 本周薄弱点 | 找出本周录音中卡壳的话题，逐个重练。**卡壳>3 次的标记为「重点薄弱」** |

### 第 1 周完成标准

- [ ] 手写 Promise 闭眼能通过
- [ ] 「输入 URL 到页面展示」3 分钟不卡壳
- [ ] Vue3 响应式链路图不翻资料能画出来
- [ ] TCP 三次握手时序图默画正确
- [ ] 防抖节流能写出 leading/trailing 全模式的完整版（10 分钟内，不是简化版）
- [ ] 「GET vs POST + HTTP 缓存字段优先级」30 秒不卡壳

## 第 2 周：项目 + 回答稿（最强的差异化）

### 前 3 天：打通项目实战

| 天 | 练什么 | 怎么练 |
|:---:|------|------|
| 1 | **项目介绍** | 用 [P0 项目实战](./阅读指南.md#项目实战与表达你最值钱的差异化) 6 个核心文件，写成 1 分钟和 3 分钟两个版本的项目介绍稿——**必须录音回听至少 3 遍**。**新增高频**: 前端重新部署后怎么通知用户刷新（version.json 轮询方案） |
| 2 | **权限 + 认证** | 脱稿讲：RBAC 模型怎么设计的（用户→角色→权限三张表）、Token 刷新怎么做到无感（双 token + 401 拦截 + 并发锁）、动态路由 addRoute 完整流程（登录→拿权限→生成路由→addRoute→挂载） |
| 3 | **Axios + 工程化** | 脱稿讲：Axios 封装做了哪些（拦截器/重试指数退避/去重/错误码映射/幂等键）→ 为什么这么设计 → **纠正**: 指数退避口径 1s→2s→4s，不是 2s→4s→8s → 不用 Axios 能换成什么 |

### 中间 2 天：面试回答逐篇练

| 天 | 练什么 |
|:---:|------|
| 4 | [面试回答/JavaScript/](面试回答/JavaScript/closure.md)——12 篇 JS 回答稿，每篇 30 秒版本和 2 分钟版本都练到不用看稿。重点练：**闭包 this、Promise 调度、Event Loop 追问** |
| 5 | [面试回答/Vue3/](面试回答/Vue3/reactivity.md) 8 篇 + [面试回答/项目/](面试回答/项目/permission-rbac.md) 4 篇。重点练：**响应式原理追问链、Diff 完整流程、项目权限设计** |

### 后 2 天：手写 + 模拟

| 天 | 练什么 | 怎么练 |
|:---:|------|------|
| 6 | **手写题 P0** | 在空白文件里限时 10 分钟写：Promise / bind / 深拷贝 / EventEmitter / debounce-throttle。**超时 = 不合格，重写直到全过** |
| 7 | **模拟面试** | 对着 [round-1-vue](模拟面试/round-1-vue.md) 自己过一遍——开录音，假装对面坐着面试官。录完回听 |

### 第 2 周完成标准

- [ ] 1 分钟和 3 分钟项目介绍各录音 3 遍，最后一版不卡壳
- [ ] RBAC + Token 刷新 + 动态路由能一口气讲完（不翻资料）
- [ ] JS 12 篇 + Vue3 8 篇 + 项目 4 篇回答稿——80% 能脱稿复述
- [ ] P0 手写题全部在 10 分钟内无 bug 通过
- [ ] Pinia storeToRefs vs 直接解构的区别能脱稿讲清（为什么解构丢响应式）

## 第 3 周：知识链 + 手写 + 查漏

### 前半周

| 天 | 练什么 | 怎么练 |
|:---:|------|------|
| 1 | **知识链 8 条** | 白纸画全部 [8 条知识链](./阅读指南.md#-跨模块知识链p0-核心任务)——每条链的箭头旁写一句因果关系。**缺箭头的链 = 你的薄弱模块** |
| 2 | **手写题 P1** | EventEmitter / LRU / 并发控制 / 列表树互转 / 版本号比较 + CSS 手写: 水平垂直居中 5 方案、圣杯/双飞翼布局——每道限时 10 分钟 |
| 3 | **P1 模块速览** | [阅读指南 P1](./阅读指南.md#-p1高频考点占比建议-25-30) 内容——只看每篇的「一句话总结」和「易错点」，3-5 分钟一篇，不逐字读。新增必看: HTML5 新特性总览（面经最常见的开场题）、前端架构 Monorepo 选型 |
| 4 | **算法 Top 5** | [面试题库 算法](./面试题库/算法.md) Top 5——反转链表、快排、两数之和、二叉树遍历、有效的括号——每道三遍：看思路→自己写→计时写 |

### 后半周

| 天 | 练什么 | 怎么练 |
|:---:|------|------|
| 5 | **HR 准备** | [自我介绍](HR/self-intro.md) 1 分钟 + 3 分钟 + [离职原因](HR/leave-reason.md) + [薪资谈判](HR/salary-negotiation.md) 话术——全部录音回听 |
| 6 | **全真模拟** | 找同事做一场全真模拟：JS → Vue3 → 项目 → HR，中间不中断。找不到人就自己对着 [模拟面试](模拟面试/) 开录音 |
| 7 | **休息** | 不学新东西。只回顾本周卡壳点——那些标记了「重点薄弱」的话题 |

### 第 3 周完成标准

- [ ] 8 条知识链——白纸脱稿画，不翻资料，每个箭头旁有因果说明
- [ ] P1 手写题全部在 10 分钟内无 bug
- [ ] 算法 Top 5——每道 5 分钟内 AC
- [ ] 自我介绍 1 分钟和 3 分钟版流畅不卡壳
- [ ] 全真模拟至少完成一遍，无重大卡顿
- [ ] VR4 新特性的三个破坏性变更（createRouter 工厂函数 / `*`→`:pathMatch` / push 不再 reject）能脱稿讲清

## 每天操作模板

```
19:00–20:00  练说（看文件 → 关掉 → 录音 → 回听对比）
20:00–21:00  手写题 / 知识链 / 算法（白纸默写，不到 deadline 不看答案）

睡前 5 分钟：列出明天要练的 3 个核心话题
```

## 判断标准

| 状态 | 标准 | 动作 |
|------|------|------|
| **过了** | 30 秒内说清核心原理 + 追问能接住第一层 | 第二天不重练这个话题 |
| **没过** | 需要停下来想、看了原文才能复述 | 第二天重练同一个话题，直到过 |
| **闭眼能写** | 10 分钟内无 bug 写出 | 这道手写题从列表中移除 |
