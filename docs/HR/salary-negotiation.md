# 期望薪资与谈判

> ⭐⭐⭐⭐⭐｜难度：中级｜谈不好前面面试全白费

## 一句话总结

**报价要说区间不说具体数、最先开口的人吃亏、拿到书面 Offer 之前不要拒绝其他面试——薪资谈判的三个铁律。**

## 核心话术

### 场景 1：HR 问「你的期望薪资是多少」

**❌ 错误**：「我要 30K」「看公司规定吧」

第一个让你底牌尽出，第二个让 HR 给你定最低价。

**✅ 正确**：「我目前的薪资大概是 XX K 左右，期望能有一个合理的涨幅。同时也看岗位的整体 package——社保公积金缴纳比例、年终奖结构、是否有期权——这些综合下来我也有一个大致的预期范围，大概在 XX-XX 之间。」

关键技巧：
- 说**区间**不说具体数——给自己留谈判空间
- 区间下限 = 你真正想要的数
- 把话题引向「整体 package」而不是单纯月薪——公积金比例差 5% 一年就是几万块

### 场景 2：HR 给的报价低于你的预期

```
HR：「我们这边能给到的月薪是 25K，年终 2 个月。」
你：（心里想要 30K）
```

**❌ 错误**：「太低了，我要 30K」（把天聊死）

**✅ 正确**：「感谢您给我这个报价。25K 确实比我目前的预期低了一些——我目前的薪资已经接近这个数了。我理解每个公司有自己的薪资体系，能否帮我了解一下：

1. 公积金是按照实际工资缴纳还是最低基数？（这差好几万）
2. 试用期薪资是全额还是八折？
3. 是否有房补/餐补/交通补贴？

如果这些福利能补上来，25K 也是可以综合考虑的。如果不行的话，您看 28K 是否在预算范围内——这个数对我也是综合考虑后的底线。」

策略要点：
- **不急拒绝**——先问福利，福利好能拉平不少
- **给出理由**——不是"我要更多"，而是"我目前薪资已经接近这个数"
- **反向给区间**——把 25K→28K 的差距变成「福利能不能补上」

> **两条红线**：
> 1. 上面的 25K/28K/30K 只是演示数字——实际报价范围因城市、年份、级别差异很大，谈之前先查同城市、同级别的近期市场数据，再定自己的区间。
> 2. 涉及现薪的表述必须属实。很多公司发 offer 前会要求提供近几个月的工资流水，或用社保/公积金基数交叉核验，虚报现薪一旦穿帮基本直接出局。

### 场景 3：手上有多个 Offer，怎么 leverage

```
你：「说实话，我目前手上有另一个 Offer，那边给的 package 会比这边高一些。
但我个人更倾向咱们这边的技术栈和团队氛围——
如果薪资能再靠近一些，我会优先选择这边。」
```

**关键**：
- 不要说具体公司名和具体金额——「更高一些」就够了
- 表达意愿——「我倾向你们」让 HR 有动力去帮你争取
- **不要 bluff**——如果没 Offer 别说有。HR 可能会让你发书面 Offer 来看看

## 薪资结构暗坑

面试阶段必须问清的 5 个数字：

| 项目 | 为什么重要 | 怎么问 |
|------|----------|------|
| **公积金比例** | 5% 和 12% 一年差好几万 | 「公积金是按照全额工资交吗，比例是多少」 |
| **年终奖基数** | 「2 个月年终」可能是基本工资×2 不是总包×2 | 「年终奖的基数是基本工资还是 total package」 |
| **试用期薪资** | 八折试用半年 = 少拿一个月工资 | 「试用期多久，薪资是全额的还是打折」 |
| **期权/股票** | 口头承诺 vs 书面协议差别巨大 | 「期权是入职签还是满一年才签，行权价和最新估值是多少」 |
| **涨薪机制** | 有的公司一年两次，有的一次都没有 | 「公司的绩效评估和涨薪周期是怎样的」 |

## 面试追问

| HR 说的 | 真实含义 | 你的应对 |
|--------|---------|---------|
| 「我们薪资有竞争力」 | 我去问问能给多少 | 「方便给一个大致范围吗，我也好综合评估」 |
| 「这个岗位预算是 XX-XX」 | 预算上限就是那个数 | 「感谢告知，这个范围我了解了」——不要在电话里直接答应 |
| 「你的期望超过了预算」 | 还有可能谈，但需要理由 | 「理解。能否帮我了解一下其他福利——如果综合 package 能达到预期，也可以考虑」 |
| 「我们 offer 只给 3 天考虑」 | 催你决策 | 请求延长到一周——「我需要和家人商量一下，能否给一周时间」 |
